# Curso Laravel: desenvolvendo um CMS

Curso Laravel abordando como construir um sistema de gerenciamento para um blog.

### Aviso!

Este curso tem por objetivo geral manter o pensamento linear de desenvolvimento, isto é, implementar mudanças no código que sejam progressivas, bem estruturadas e que não quebrem funcionalidades existentes.

Cada etapa (aula) irá focar em entregas pequenas e contínuas, permitindo validação rápida e facilitando a manutenção e a escalabilidade do CMS.

Se você já tem familiaridade com outros frameworks, pode se sentir “tentado” a tomar um atalho, mas a didática empregada aqui visa explicar em detalhes como o framework Laravel interpreta diferentes modos de se realizar a mesma tarefa.
